What is MyRing?

Serving Data and content to 1.4 Billion smartphones and 7 billion connected devices it’s the biggest opportunity ever imagined. Although the lack of accessible tools to prepare, capture and share data block thousands of content creators to do so. MyRing Data Suite facilitates the data funnel in every step of the process, from capturing it all the way to delivering it worldwide via Content Distribution Networks. MyRing allows anyone with Data to make it accessible to the world and to obtain value out of it.

What is Data?

Data is any bit of unprocessed information. Normally it becomes processed information or “content” when it's assembled, combined or merged with more Data.

What is the Data Suite?

The Data Suite is a solution that helps anyone through each step of the Data cycle: Modeling, Capturing and Exposing (Sharing).

What is Modeling?

Modeling is the action of creating a "Data Schema." With modeling we organize the users data. We generate a blueprint of the information and its structure. This allows us to capture all data schemas, organizing every single detail of the data. For example there may be a data schema for “Birds Species” detailing: color, region, family, etc.. But also there may be another one detailing baseball players Information: Name, position, team, records, etc. MyRing provide tools to model Data Schemas and to tag data semantically.

What is a Ring schema?

A Ring schema is a schema created and maintained by one or more individuals. A Ring schema lives in Github and it’s an open source project. Its popularity depends on the amount of people capturing and using it.

What is Capturing?

Capturing is the action of collecting new data as opposed to scrapping, which only reprocess existing Data. Capturing is one of the most important actions of the Data cycle, doing it correctly will define Data usability and relevance. MyRing provides tools for capturing data manually (by humans) and automatically (by sensors).

What is Exposing (Sharing)?

Exposing is the action of making data available to the world to use. Exposing your data means making it accessible in any format to any system. Currently one of the most common Data Formats is JSON REST. My Ring provides tools to expose data in this and any other existing formats (SOAP, CSV, SPARQ, TOML, etc.)

What is the difference between Storing Data in the cloud and using a Ring.?

It’s a very different process. Unlike storing (Dropbox and similar) where information is held in one singe location as files, MyRing captures data resources, organize them semantically and distributes them to different systems via an API allowing anyone to use it to assemble their own content in the form of an mobile APP or a website. Data on a Ring can be used several times by many internet applications in hundred of different ways.

Can MyRing be used to serve content?

Yes absolutely, Rings can be used to serve content in the same way it’s used to distribute data. A "video" for example will be treated as a raw data source. However in MyRings the same source will also include information about Author, license, synopsis, credits, etc.

What about the licenses to use the Data I’m distributing?

Every bit of data distributed via a Ring has a license attached to it. This will help guide a system in the use of it.
