What is a Ring?

A Ring is crowd aggregated Data Set. It’s a pool of information gathered by one or more individual. A ring can be combined with other rings to create rich results.

What is Data?

Data is any bit of unprocessed information. Normally it becomes processed information or “content” when is assembled, combined and merged with more Data.

What is the Data Suite?

The Data Suite is a solution that helps you through each of the Data cycle: Modeling, Capturing and Exposing (Sharing).

What is Modeling?

Is the action of creating the Data schema that will organize the Data. A data schema is the equivalent of the DNA of organized (Semantic) data, it’s the blueprint that have detailed information on how the Data will be structured. There’s as many data schemas as areas to be captured for example there can be a data schema of “Birds” detailing: color, region, family, etc.. But also there can be another schema detailing info about baseball players: name, position, team, records, etc. MyRing provide tools to model Data schemas and to tag data semantically.

What is a Ring schema?

A Ring schema it’s a Data Schema created and improved by one or more individuals. A Ring schema lives in Github as an Open source project and it’s popularity depends on the amount of people capturing and sharing Data based on it.

What is Capturing?

Is the action of collecting new Data as opposition to scrapping which only reprocess existing Data. Capturing is one of the most important action of the Data cycle and doing it correctly will define it’s usability and relevance. MyRing provides tools for capturing Data manually (by humans) and automatically (by sensors).

What is Exposing (Sharing)?

Is the action of making Data available to the world to use it. And this goes literally, exposing your data means making it accessible in any format that and device or system understands. One of the most common Data Formats nowadays is JSON REST, My Ring provide tools to expose Data in this format and any other existing one (SOAP, CSV, SPARQ, TOML, etc.)

What is the difference between Storing Data in the cloud and using a Ring.?

It’s very different story, not even in the same category. Dropbox and similar services store and share unrelated static files. While MyRing is more about capturing fresh data and distributing it to different systems that will use it to assemble their own content in the form of an APP or a website. Data on a Ring can be used several times by many internet applications in hundred of different ways.

Can MyRing be used to serve content?

Yes absolutely , Ring can be used to serve content the same way it’s used to distribute data, A video for example will be treated as a Raw Data Resource, additional Data in the Ring will have information about Author, license, synopsis, credits, etc.

What about the licenses to use the Data I’m distributing?

Every bit of data distributed via a Ring have a license attached to it. This will help any system to understand what is the best way to use it.
